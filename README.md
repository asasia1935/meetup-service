# Meetup Service

관심사와 지역 기반의 커뮤니티/모임 플랫폼을 위한 백엔드 서비스입니다.

---

## 🎯 Goal

이 프로젝트는 단순한 일정 관리 서비스가 아니라,

사용자가 관심사 기반으로 커뮤니티에 참여하고  
모임(Meetup)을 생성 및 참여하며  
오프라인 활동 이후 다시 온라인 상호작용으로 이어지는  

**커뮤니티 중심 플랫폼**을 구현하는 것을 목표로 합니다.
